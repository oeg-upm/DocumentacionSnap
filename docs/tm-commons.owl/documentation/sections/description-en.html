This module focuses on the representation of information about the common concepts. As such, the main types of classes that are represented in this ontology are:
  • Group of Entities, a set of Entities grouped together according to a Purpose of Grouping, e.g. grouping of stops known to the public by a common name.
  • Presentation, used to represent a Line, or any other object, in a map.
  • Zone, a two dimensional Place within the Service Area of an Operator or Serviced Operator.
  • Place, a geographic place of any type which may be specified as the origin or destination of a trip.
There is also one reference SKOS Concept Scheme to refer to the types of organisations that can be described using this ontology.
