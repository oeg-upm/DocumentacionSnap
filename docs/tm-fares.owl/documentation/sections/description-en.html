Transmodel Ontology - Fares: Description back to ToC

This module focuses on the representation of information about the fares. As such, the main types of classes that are represented in this ontology are:
  • BookingArrangements, created for FlexibleLine
  • ClassOfUse, a classification of fare and other service classes by category of user entitled to use them.
  • DistributionChannel, which indicates the type of outlet for selling a Fare Product
The main types of properties that are represented in this ontology are:
  • distributionChannelType, the property that assigns to a DistributionChannel one or several SKOS Concepts from the codelist on types of distribution channels.
  • gobernedBy, the property that relates a Distribution Channel to the Organisation that takes responsability over it.
    • There is one reference SKOS Concept Scheme to refer to the types of fares that can be described using this ontology.
